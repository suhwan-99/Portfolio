# 기술 스택 로고 이미지 폴더

이 폴더에는 프로젝트에서 사용하는 기술 스택의 로고 이미지를 저장합니다.

## 이미지 파일 형식

- SVG 형식을 권장합니다 (확장성과 선명도)
- PNG 형식도 가능합니다
- 파일명은 소문자로 작성하고 하이픈(-)으로 구분합니다
  - 예: `react.svg`, `spring-boot.svg`, `raspberry-pi.svg`

## 필요한 로고 이미지

다음 기술 스택들의 로고 이미지를 추가해주세요:

- React (`react.svg`)
- Redux (`redux.svg`)
- WebSocket (`websocket.svg`)
- MQTT (`mqtt.svg`)
- Raspberry Pi (`raspberry-pi.svg`)
- Python (`python.svg`)
- SCSS (`scss.svg`)
- CSS Modules (`css-modules.svg`)
- Intersection Observer API (`intersection-observer.svg`)
- Spring Boot (`spring-boot.svg`)
- JPA (`jpa.svg`)
- MySQL (`mysql.svg`)
- VS Code (`vscode.svg`)
- Cursor (`cursor.svg`)

## 로고 이미지 다운로드

로고 이미지는 다음 사이트에서 다운로드할 수 있습니다:

- [Simple Icons](https://simpleicons.org/)
- [DevIcon](https://devicon.dev/)
- [Iconify](https://iconify.design/)

## 사용 방법

이미지 파일을 추가한 후, `src/data/techLogos.js` 파일에서 경로를 확인하세요.
Vite를 사용하는 경우, 이미지를 import하거나 public 폴더를 사용할 수 있습니다.
